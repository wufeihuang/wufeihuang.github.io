使用d3.js实现地图（二） - 基于TopoJSON优化

发表于 2019-04-07 | 更新于 2019-04-10 | 分类于 d3js

上一篇 提到了用GeoJSON格式数据绘制地图有缺点，这一篇就探讨如何基于TopoJSON来优化。

TopoJSON介绍

TopoJSON是使用拓扑结构编码来实现的一个GeoJSON的扩展。官方原文是这样的：”TopoJSON is an extension of GeoJSON that encodes topology.”具体的介绍可以看官方文档。

总之TopoJSON相对于GeoJSON有很大优势：消除了大量冗余（比如边界地理数据重复等），使得描述同一地理的TopoJSON文件比GeoJSON文件体积小了很多；就绘制而言，使用TopoJSON可以做到仅绘制外围轮廓、共享边境不重复绘制、查找相邻地区等。

TopoJSON已经被广泛使用了，而且由于是由D3js的作者Mike Bostock设计开发的，所以D3js社区有大量的使用TopoJSON绘制地图的案例。

topojson的使用

TopoJSON和GeoJSON一样，本身只是一种文件格式、一种地理数据标准。而为了对TopoJSON数据做操作，需要用到一个叫 topojson 的js库。GitHub地址

topojson又分为topojson-server、topojson-simplify和topojson-client：topojson-server用于将GeoJSON转换成TopoJSON；topojson-simplify用于定制规则以缩小TopoJSON文件体积；topojson-client用于将TopoJSON转成GeoJSON，并且提供了对应之前提到的TopoJSON在绘制上的优势的API。顺带提一下，除了像普通js库一样使用，topojson还可以用于命令行。

绘制时常用API

要注意的是地图绘制都是基于GeoJSON的，TopoJSON并不能直接用于绘图，它总是要转换成GeoJSON格式才能被用于绘图。

topojson.topology((objects[, quantization])

将GeoJSON格式的数据对象转换成TopoJSON格式的对象。

第一个参数是包含GeoJSON数据的对象，第二个可选参数是指转换过程中处理坐标时的精度。

要注意的是第一个参数并不能直接传GeoJSON数据作为参数，而是需要这种key-value的形式，这个key会被用于在返回的topology对象中进行索引，具体使用看后面。

一个返回的topology对象（TopoJSON对象）通常包含以下几个属性：

  • arcs
  • bbox
  • objects - 几何对象集合
  • transform
  • type: “Topology”

topojson.feature(topology, object)

将TopoJSON对象中的几何对象转换成GeoJSON，返回的是Feature 或 FeatureCollection几何对象。

第一个参数是要转换的TopoJSON对象，第二个参数是该对象内具体的某个几何对象。

返回的这个geo_data就能当成普通的GeoJSON数据来使用。

topojson.merge(topology, objects)

简单点说就是获得objects包含的几何体组成的整体的外轮廓。返回的是一个GeoJSON的MultiPolygon几何对象。

topojson.mesh(topology[, object[, filter]])

获得objects构成的整体的完整轮廓线，包括外轮廓和内部共享边界线，但是是一个整体的线，所以没有线条重复的情况。返回的是一个GeoJSON的MultiLineString对象。

如果不传objects参数，那就是处理整个topology。

可选的filter参数是一个函数，它又接收两个参数。这个filter参数通常用来过滤不共享的外轮廓或者被共享的内部边界线。

a !== b会过滤掉外轮廓；a === b则会过滤掉内部被共享的线。

topojson.neighbors(objects)

会返回一个数组，数组中的子数组对应着objects中相应索引的几何对象的相邻几何对象，但是只是包含相邻对象的索引。

topojson虽然帮我们解决了避免相邻边界重复绘制以及获取整体外轮廓的问题，但是可以看到，这二者各自都是完整的路径对象，所以当要在鼠标悬浮时变换地区边框颜色这样的功能就做不到了。因此需要一层额外的遮罩层或者叫交互层，它就和上一篇单纯基于GeoJSON实现的地图是一样的，但是将这一层设为透明，仅当鼠标悬浮时显示对应的区域，对这一区域可以设置独立样式，因为它恰好覆盖了底下的图层，所以就达到了我们需要的效果。
