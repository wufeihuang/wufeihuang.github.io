前言

杭州这边前端招聘大多是要求React栈，所以学习下React，对比学习应该也能更有助于理解三大框架的原理和意义。

暂时先根据FreeCodeCamp的React教程学习。之前刷过一些基础内容，这个网站用于入门还是很不错的。

这一篇的内容主要就从这个课程的内容摘选并翻译过来。

React介绍

React 是由 Facebook 推广开来的，一个用于创建用户界面的开源JavaScript库。它用于创建组件、处理状态和属性，利用事件监听器和某些声明周期方法来更新数据。

It is used to create components, handle state and props, utilize event listeners and certain life cycle methods to update data as it changes.

React结合HTML和Javascript的能力创建了他自己的标记语言——JSX。

JSX

JSX是JavaScript的一种语法扩展，它允许你直接在JavaScript中写HTML。

这有几点优势：它使得你可以在HTML中充分利用JavaScript的编程能力，并保持你的代码可读性。

JSX大部分情况和HTML一样，但也有一些不同：比如可以直接在JSX中写JavaScript代码，具体的用法是使用大括号将JavaScript代码括起来。

编译

由于JSX并不是合法的JavaScript，所以JSX代码必须被编译成JavaScript，其中常用的编译工具便是Babel。

单个根元素

关于JSX很重要的一点是JSX可以像HTML一样嵌套，但必须返回单个元素，即用一个根元素包裹住所有其他元素；否则会无法编译。

使用括号

可以用括号将JSX代码括起来，但这并不是严格必要的。

注释

JSX中使用{/* comments */}的语法来添加注释。

定义HTML的class以及camelCase命名

JSX中写HTML时不能使用class这个词，因为class是JavaScript的保留字，所以在JSX中使用className替代。

另外JSX中所有的HTML属性与事件引用都必须使用camelCase规则，比如onclick要写成onClick、z-index要写成zIndex。
